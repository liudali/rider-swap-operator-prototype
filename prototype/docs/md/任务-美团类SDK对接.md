# 任务：美团类 SDK / 第三方渠道包对接 · 已取消

> **状态**：**已取消**（2026-06-12 产品拍板）  
> **决议**：去除美团 / 第三方 App SDK 订单推送；**不纳入一期及近期路线图**  
> **替代方案**：**骑士卡差价模式** — CO 批发购卡 → 卡库存 → 持卡骑手 → 平台代收零售 + 对账单差价线下结（C-06/C-08）
