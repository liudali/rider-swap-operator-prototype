| 结论 | **已锁定**：凡费用明细列表统一提供 **类型** 与 **时间（日起/日止）** 筛选项 |

## 覆盖范围

| 明细 | 类型筛选项 | 时间 |
|------|------------|------|
| 跨网服务费明细 | 柜机费 / 电池费 | 日起止 + 快捷范围 |
| 日清账单 | 扣款来源（保证金/信用额度） | 日起止 + 快捷范围 |
| 周月汇总 | 周 / 月 | 周期日起止 |
| C 端支付分账 | 包月 / 次卡 | 支付日起止 |
| B 端消耗计提 | 确认消耗 / 激活码核销 | 日起止 |
| 保证金变动明细 | 对公充值 / 日清划扣 / 平台费代扣 | 日起止 |
| 平台保证金账本 | 同上 + 手工入账 | 日起止 + 运营商 |
| 清分明细 | 支付清分 / 退款冲正 | 关联支付日起止 |
| 平台提成流水 | C/B 触发类型 | 日起止 |

## 支持理由

- **俞军**：对账场景「按类 + 按日」是高频路径，缺一不可
- 筛选条复用既有 `#pageFilters` / `pfKey`，无新导航噪音

## 失效条件

- 某明细无业务类型维度 → 类型可改为「全部」占位或隐藏（本期仍保留）
