# 决策卡片 031 · 跨网服务费明细并入运营商往来页内 Tab

| 项 | 内容 |
|----|------|
| 日期 | 2026-07-16 |
| 阶段 | MVP / 信息架构 |
| 结论 | **已锁定**：侧栏去掉独立「跨网服务费明细」L2；并入「运营商往来」页内 Tab（概况 / 跨网服务费明细） |

## 支持理由

- **乔布斯减法**：概况 KPI 与明细同属「看往来」一条任务，拆两个 L2 增加侧栏噪音
- **俞军价值公式**：从趋势下钻到明细零跳转成本（页内 Tab）

## 反对理由

- 页内 Tab 不如侧栏 L2 可被权限单独挂载 `[假设]` 仅 `inter_op.view` 即可

## 失效条件

- 明细需独立深链/权限与概况完全拆开 → 可再升 L2
