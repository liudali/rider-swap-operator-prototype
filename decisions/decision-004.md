# 决策卡片 004 · 运营商概览 KPI 合并

| 项 | 内容 |
|----|------|
| 日期 | 2026-07-15 |
| 阶段 | 原型迭代 / 发布前打磨 |
| 结论 | 经营概览设备区改为「柜机 / 电池 / 站点」三卡；财务提现与跨网等从总览下线 |

## 决策

1. **柜机** = 自有 + 租赁，统一在线率  
2. **电池** = 自有 + 租赁，统一在线率（柜内随柜机；柜外按 health=正常 [假设]）  
3. **站点** 替代「渠道商」，副文案为营业状态（在营 / 建设中 / 停用）  
4. 下方经营指标仅保留「套餐购买金额」「活跃用户」；去掉可提现、提现待审、跨网待日清、平台服务费及顶部设备拆分 Banner

## 支持理由

- 俞军：总览要答「现在能不能开店/运营」，自有/租赁拆分增加认知成本，替换价值为负  
- 乔布斯：总览做减法；渠道签约、清分提现有独立页面，不必抢第一屏  

## 反对理由

- 运营若需快速看「租赁占比」，总览不再一眼可见 → 需下钻设备列表  
- 「套餐购买金额」≠ 清分入账，可能被误解为可提现 → 文案已标注「C 端实付」

## 失效条件

- 一期验收明确要求总览展示租赁占比 / 可提现余额  
- 电池 IoT 独立 online 字段上线后需替换当前在线判定假设

- 正式环境电池在线规则与研发对齐后更新 `isBatteryOnline`
