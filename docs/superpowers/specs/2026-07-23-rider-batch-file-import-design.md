# 骑手批量导入 · 文件方式（含 xlsx）

日期：2026-07-23 · 决策：[decision-063](../../../decisions/decision-063.md)

## 目标

渠道商人天额度池「骑手登记 → 批量导入」在保留粘贴录入的同时，增加文件导入（CSV / TXT / XLSX）。

## 交互

1. 弹窗顶部：所属团队（共用）
2. 导入方式：`手工录入（少量）` | `文件导入（大量）`
3. 手工：多行 `手机号,姓名`
4. 文件：上传；选文件后展示「已解析 N 条」；下载 **CSV 模板** / **Excel 模板**
5. 保存：走既有校验（个人套餐 / 已在职 → 失败清单）

## 文件约定

| 项 | 规则 |
|----|------|
| 格式 | `.csv` / `.txt` / `.xlsx` |
| Sheet | xlsx 取**第一个**工作表 |
| 列 | 手机号、姓名（首行可为表头） |
| 分隔 | CSV/TXT：逗号 / 制表符 / 中文逗号 |

## 原型实现

- 文本：`FileReader.readAsText`
- xlsx：动态加载 SheetJS（CDN）解析 `ArrayBuffer`
- 解析结果缓存在提交前变量，与设备 SN 导入同构

## 非目标

异步任务中心、多团队列、`.xls`
