# 平台用户信息 · 押金与持有电池列（设计）

**日期**：2026-07-24 · **决策**：[decision-066](../../../decisions/decision-066.md)

## 范围

平台管理员 → 用户管理 → 用户信息列表。

## 列定义

1. **电池押金**：方式 + 明细（实付实收 / 信用分 / 渠道名 / ——）
2. **押金状态**：**仅实付** → 在押 / 退押中；信用免押、渠道担保、无记录 → ——
3. **持有电池**：`编码-SOC-SOH（归属运营商）` 或 `未持有`

## 筛选

电池押金与押金状态分筛；「退押中」不再挂在电池押金筛选项下。

## 非目标

运营商后台「用户」列表本期不强制同步拆列（展示可复用新 methodText）。
