# 验收标准 · 运营商/代理商后台（原型）

## 通用

- `python3 main.py` 端口 `8766` 可预览
- 侧栏展示当前登录主体（演示：运营商名称）
- 可切换演示角色：一级代理 / 二级代理 / 合作运营商 / **租赁公司**；导航随角色变化
- 合作运营商：**总览 / 站点 / 我的设备 / 我的订单 / 我的流水 / 员工 / 用户 / 合作伙伴分润 ★**
- 一二级代理：**总览 / 组织管理 / 员工 / 我的设备 / 我的订单 / 我的流水 / 下级分润 ★**
- 租赁公司：**总览 / 员工 / 协议与设备 / 租金收缴 / 收款账户**
- 侧栏「切换登录身份」含：经营主体、员工登录、合作伙伴登录
- 切换一级代理：仅见华东一级自有设备订单（如徐汇点）；闪送驿站订单不在此列
- 切换二级代理：仅见浦东二级自有设备订单；闪送/陆家嘴订单在「下级分润」
- 无全局筛选条；各页面独立筛选区，「重置筛选」仅重置当前页
- 模块「说明」弹窗可用

## 模块

| 模块 | 验收项 | 预期 |
|------|--------|------|
| 总览 | KPI | 在营站点、柜机在线率、订单数、净入账、活跃用户 |
| 总览 | 站点分析 | 表格按站点展示用户数、换电数、订单总金额；随页面「站点/统计范围」筛选；含合计行 |
| 租赁 | 协议与设备 | 协议表展示绑定设备；出租方可「新增协议」；抽屉含设备清单 |
| 租赁 | 租金收缴 | 租赁公司独立菜单：账期、应收/已收/待收、收缴方式、待跟进 |
| 租赁 | 月租金 | 承租方：设备分摊租金、收入覆盖、自动扣款设置、手动缴纳/处理缺口 |
| 租赁 | 收益不足 | 闪送驿站 2026-06 演示缺口 ¥1000 + 橙色提示条 |
| 账户 | 收款账户 | 各主体均有「收款账户」菜单；展示通道、商户号、用途 |
| 员工 | 菜单 | 一/二级代理、合作运营商、租赁公司均有「员工」 |
| 员工 | 运营员工 | Tab「运营员工」；新增/编辑；勾选功能权限（总览、订单、设备等） |
| 员工 | 合作伙伴 | Tab「合作伙伴」；维护名称、联系人、分润比例、关联站点 |
| 登录 | 运营员工 | 如「李小运维」：仅见设备/订单/用户/站点等已授权菜单 |
| 登录 | 合作伙伴 | 如「浦东驿站物业合作方」：个人账户 + 分润明细（仅关联站点） |
| 个人账户 | 余额/提现 | 可提现余额、银行卡、提交提现、提现记录 |
| 个人账户 | 提现演示 | 提交后扣减可提现余额，记录状态「处理中」 |
| 站点 | 列表 | 运营商可见当前合作方（只读）；「分润设置」跳转分润页 |
| 合作伙伴 | 分润设置 | 下拉选项来自「员工-合作伙伴」；可为每站点选择合作方 |
| 合作伙伴 | 订单明细 | 关联站点套餐/换电；用户 ID/手机完整展示 |
| 合作伙伴 | 结算 | 分润结算批次表 |
| 总览 | 运营商 KPI | 「合作伙伴待付」 |
| 设备 | Tab+列表 | 换电柜 / 电池；编号、站点、状态 |
| 订单 | Tab | 套餐购买订单 / 换电订单 / **服务变更** |
| 订单 | 筛选 | 套餐：单号、手机、支付日期、服务/订单状态；换电：单号、手机、换电日期、状态 |
| 订单 | 中途完结 | 演示 SUB260523088：未使用套餐+押金退款拆分；资金实收含押金退还流水 |
| 订单 | 冻结解冻 | 演示 SUB260524002 已冻结；解冻申请可「通过解冻」恢复服务 |
| 订单 | 免押 | 演示芝麻信用、微信支付分免押用户 |
| 订单 | 押金 | 套餐列表与详情展示押金/免押状态 |
| 订单 | 关联 | 换电列表展示关联套餐单号，可点开套餐详情 |
| 订单 | 套餐详情 | 有效期、期内使用、累计应分、打款状态 |
| 订单 | 换电 | 表格展示主要字段（用户、站点、柜机、格口、电池、应分）；点击详情打开抽屉 |
| 订单 | 换电日志 | 详情抽屉内时间线：用户操作、请求、设备动作/反馈；失败单含超时链路 |
| 组织 | 权限 | 一级不可见/编辑二级下属合作运营商（如闪送驿站联营） |
| 组织 | 一级 | Tab 二级代理/直属合作运营商；新增、编辑、分成比例 |
| 组织 | 二级 | 直属合作运营商列表；新增、编辑、分成比例 |
| 设备 | 隔离 | 仅列表展示当前主体自有设备 |
| 订单 | 隔离 | 仅展示 device_owner_id=当前主体的套餐/换电 |
| 下级分润 | 订单明细 | 逐笔套餐单；含计提金额；可按下级 Tab 筛选 |
| 下级分润 | 脱敏 | 仅显示脱敏用户 ID/手机；无姓名、无跳转骑手 |
| 下级分润 | 模块 | 订单明细置于汇总表之前；含结算批次 |
| 流水 | Tab | 资金实收 / 应分台账 / 打款批次 |
| 流水 | 资金实收 | 套餐支付、退款出款；实收/手续费/入账/通道 |
| 流水 | 应分台账 | 仅合作运营商/二/一级应分；无平台抽成；含冲正与周期汇总 |
| 流水 | 架构B | 资金实收展示收款主体/商户号；打款批次为向上分账 |
| 流水 | 打款批次 | 周期完结后从合作运营商商户向代理分账；进行中不划出 |
| 订单 | 套餐 | 合作运营商视角展示收款主体（架构B） |
| 用户 | 列表 | 用户、绑定站点、套餐、最近换电 |
